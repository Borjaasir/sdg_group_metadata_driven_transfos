The information in the output field in metadata.json means the following:
The input field is the name of the transformation dataset.
The type field is either file or table, for file persist the file and for table create a table using a connection to a duckdb database.
The config.table is the name of the table in the database if the type field is table
The config.format is the format output file, it can be parquet o csv if the type field is file
The config.save_mode is the data operation strategy to perform, it can be overwrite, append or merge
The primary_key is the key or composite key for table types
The partition is to partition the data using that column
